# DeepState
DeepState

DeepState is a productivity and focus-tracking web application designed to help users manage their mental load, track focus sessions, and maintain streaks—all while gamifying productivity. Inspired by apps like Duolingo, DeepState emphasizes simplicity, clarity, and user engagement.

Features

Focus Timer
Start focus sessions based on Cognitive Load Mode (Light, Medium, Heavy). Timer starts at 00:00 and counts down the selected duration.

Cognitive Load Mode (CLM)
Select different modes for tasks depending on mental intensity:

Light (25 min)

Medium (45 min)

Heavy (90 min)

Focus Debt System
Abandoned sessions add “Focus Debt,” encouraging users to maintain consistency and complete sessions.

Streak & Daily Goal Tracking
Track consecutive days of focus and daily focus minutes to maintain momentum.

History Tab
Scrollable history of past sessions with task names, duration, mode, and completion status.

Onboarding Screens
A guided introduction for first-time users, explaining the app, CLM, and focus debt.

Reset App Button
Clear all progress and start fresh for testing or new users.

Tech Stack

HTML5 – Structure of the app

CSS3 – Responsive, modern styling and animations

JavaScript – App logic, timers, state management, and localStorage persistence
